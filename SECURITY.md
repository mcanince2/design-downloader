# Security Policy

## Reporting a Vulnerability

Please report security issues **privately** by emailing **mcan.ince3@gmail.com**
rather than opening a public issue. You'll get a response as soon as possible.

## Secrets & API keys

Design Downloader requires **no API keys** and ships with no credentials. If you
fork it and later add a service that needs a key, keep it in a local `.env`
(git-ignored) and never commit it or paste it into issues/PRs/logs.

## Responsible use

This tool is for downloading content you own or are licensed to use, and for
personal archiving. Respect each platform's Terms of Service and creators'
copyright.

## Supported Versions

The latest release on the `main` branch is supported.
