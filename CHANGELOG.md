# Changelog

All notable changes to this project are documented here. The format is based on
[Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [1.0.0] - 2026-06-02

### Added
- Initial public open-source release.
- Unified downloader for Behance, Dribbble and Pinterest.
- Desktop app (CustomTkinter) with a live download queue.
- Local Flask API server backing the Chrome extension.
- Chrome extension (Manifest V3): popup, right-click menu, current-page download.
- AI-assisted folder organization.
- Requires no API keys.
